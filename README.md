# Doc Taco Academy — GitHub Practice

Welcome! This little repository is your **practice sandbox** for learning
GitHub itself. It is a complete, independent copy that belongs to you —
nothing you do here can affect anyone else, so experiment freely.

You'll use it twice in orientation:

- **O-A.2 — GitHub in the browser:** edit files, make commits, create a
  branch, and open a pull request, all on github.com.
- **O-A.5 — Git on the command line:** clone this same repo in your
  codespace and do the daily loop — `status`, `add`, `commit`, `push`.
